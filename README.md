# Trading Systems

Trading systems with cTrader integration, SQL Server database, and Python strategies.

## Structure

- `ALGOs/` - cTrader trading bots (C#)
- `PythonScripts/` - Python trading strategies and utilities  
- `SQL/` - Database schema, stored procedures, functions
- `Backups/` - Backup scripts and configurations
- `Documentation/` - System documentation

## Setup

1. Database: SQL Server with cTrader schema
2. cTrader: Connect bots to trading platform
3. Python: Install requirements from `requirements.txt`

## Maintenance

- Automated database backups
- Index maintenance
- Statistics updates
- Integrity checks

## Security

- Database credentials in environment variables
- Encrypted connections
- Regular security audits
